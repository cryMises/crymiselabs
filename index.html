WhatsApp AI & Tools Chatbot
A full-featured AI chatbot for WhatsApp powered by a locally-hosted LLM via Ollama. Supports natural conversation, tool-calling (weather, web search, reminders, calculator), per-chat session memory, and multi-user handling — all running on self-hosted infrastructure with zero cloud AI dependencies. Integrated with n8n for workflow automation triggers and built with the official WhatsApp Cloud API.
Python Ollama WhatsApp Cloud API n8n Tool Calling
